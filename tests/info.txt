test_1: Busca la solución a un laberinto suministrado por la entrada estándar. 
    Se puede sumintrar desde un fichero redirigiendo la entrada estándar.
    El fichero debe contener en la primerwa linea la cantidad de filas que contiene el laberinto y a continuación la cantidad de columnas.
    El laberinto tiene el formato con el siguiente código:
        '#' indica que es una pared del laberinto y por consecuente, no puede pasar por ahí.
        ' ' es un hueco por el que puede pasar.
        'o' indica que ya ha pasado por ese camino. 
    Al ejecutar el laberinto, va marcando con 'o' por dónde ha pasado.
    La busqueda del camino finaliza cuando llega al hueco que debe haber en el hueco de abajo a la derecha.
    En el caso de que no encuentre ningún camino para resolver el laberinto porque no hay hueco abajo a la derecha o porque no hay un camino para llegar hasta ahí, indica un mensaje.
    Un ejemplo de la entrada esperada es:

        11 20
        ###################
        #             # # #
        ########### # # # #
        # #       # # #   #
        # ######  # ##### #
        # #    #  #     # #
        # # ####  # ##### #
        # #    #          #
        # #  # ########## #
        # #               #
        ###################
    
    El programa muestra durante la búsqueda, el camino por el que pasa, indicando con coordenadas,la posición por la que está buscando el camino y el retroceso que realiza en la búsqueda si en camino no es correcto.
